# Delanta test

## Project setup

### Node.js Server
```
cd node-express-server
npm install
node server.js
API will be available in `http://localhost:8080/`.
```

### Angular Client
```
cd angular-14-client
npm install
npm install -g @angular/cli
```
Run `ng serve --port 8081` for a dev server. 
Navigate to `http://localhost:8081/`.
